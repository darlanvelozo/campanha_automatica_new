💡 Como funciona o Template SQL:

O SQL será executado automaticamente para buscar clientes com email válido. Todos os campos retornados pela consulta estarão disponíveis como variáveis no template de email.

Os dados já processados desta execução serão reutilizados. Isso é mais rápido pois não precisa executar o SQL novamente.
